クラスの書き方
class (クラス名)
{

    メンバ変数（フィールド）

    コンストラクタ※

    メンバメソッド

    (プロパティ)←ここでは特に触れない

}


class (クラス名)
{
    インスタンス

    メンバメソッドの呼び出し

    etc...

}

※コンストラクタ
メンバ変数で定義したものは、null(なんも中身がない)なので、初期の値を定義するもの

・クラス内で宣言するものは、メンバ(ほにゃほにゃ)となる
・メンバ(ほにゃほにゃ)はアクセス修飾子を付ける
　　アクセス修飾子 ほにゃほにゃ 
    　　public     baseAttack = 20;
・this.(変数名)
    メンバ変数の呼び出し文法（他のメソッドとかでメンバ変数使う時に、用いるもの）

---実例---
